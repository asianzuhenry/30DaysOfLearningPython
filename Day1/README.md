# Print, Variables

## Print
- Used to output text to the console
- Syntax: `print("text")` or `print(variable)`
- Can print multiple items separated by commas
- Example: `print("Hello", "World")`

## Variables
- Containers for storing data values
- Created by assignment: `variable_name = value`
- Naming rules: start with letter/underscore, alphanumeric + underscore only
- No type declaration needed (Python infers type)
- Examples:
    - `name = "Henry"`
    - `age = 25`
    - `is_learning = True`

## Common Patterns
- `print(variable_name)` - display variable content
- Combine with f-strings: `print(f"Hello {name}")`
