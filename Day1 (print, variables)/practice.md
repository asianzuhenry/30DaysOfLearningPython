# Practice Questions

1. Write a Python program that stores your name in a variable and prints "Hello, \<name\>!" using that variable.

2. Create two variables, `a` and `b`, assign them integer values, and print the result of adding them together.

3. Define a variable `message` containing a string and use `print()` to display it.

4. Use `print()` to display the value of a variable and a literal string on the same line.

5. Assign a floating-point number to a variable and print it with a descriptive label.
